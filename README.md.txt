Для поиска минимального значения используется функция minimum().
Она принимает список в качестве единственного аргумента.

Для сохранения найденного значения используется переменная min_elem,
которой изначально присваивается первый элемент списка.

В цикле for каждый элемент сравнивается с этой переменной.
Если он меньше min_elem, то сохраняется значение этого элемента в нашей переменной.
После сравнения со всеми членами списка в min_elem гарантировано
находится наименьший элемент.

С клавиатуры вводится число элементов массива n, далее вводятся сами элементы.
